LeetCode: Remove Nth Node From End of List

Posted on 2018-06-27
Given a linked list, remove the n-th node from the end of list and return its head.

题目

Given a linked list, remove the n-th node from the end of list and return its head.

Example:

Note:
Given n will always be valid.

Follow up:
Could you do this in one pass?

翻译

给一个链表，从链表的末尾移除第n个节点并返回头节点。假设n总是合法的。尝试在一次遍历中解决问题。

思路

这个问题的难点在于如何找到链表中的倒数第n个节点。同时题目要求一次遍历就能解决这个问题，这就意味着不能先遍历一遍确定链表的节点个数后再移除，而是需要在遍历到对应节点的时候就移除。
这道题目可以用一个巧妙的想法快慢指针去解决，先让快指针往前走n步，这样快慢指针就相差n个节点，再让快慢指针同时走，直到快指针指向最后一个节点时，慢指针指向的就是倒数第n个节点的前一个节点。

n=2时

  • 1 <- slow/head
    2
    3 <- fast
    4
    5

  • 1 <- head
    2
    3 <- slow
    4
    5 <- fast

这里还有个问题是，有可能n就是链表的长度，那么在最开始快指针往前走n步的时候就已经指向最后一个节点的后面了，这个时候需要返回的是头部指针指向的下一个节点。

n=5时

  • 1 <- slow/head
    2
    3
    4
    5
    <- fast
